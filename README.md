Greencraft - JavaFX 3D demo with design patterns

How to run:
1. Install JDK 17+.
2. In IntelliJ IDEA: open the project (pom.xml).
3. Run with Maven: mvn javafx:run

Controls:
SPACE - jump
1 - create GRASS block
2 - create STONE block (decorated)
DELETE - remove last block
F1 - Walk movement
F2 - Fly movement
F3 - Slide movement

Assets folder contains example textures and sounds.
